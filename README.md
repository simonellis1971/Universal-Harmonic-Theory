Overview

Universal Harmonic Theory is an original conceptual research programme exploring harmonic organisation as a proposed fundamental principle underlying coherent physical structure. The repository contains the evolving manuscripts, supporting documents, publication material, and future revisions associated with the archived Zenodo record.

The current repository supports transparent development of the project while the Zenodo archive provides permanent scholarly preservation and citation.
